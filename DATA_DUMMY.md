## ğŸ”§ Cara Menggunakan Seed Data

### Otomatis

Data dummy akan otomatis ditambahkan saat pertama kali menjalankan aplikasi jika database masih kosong.

### Manual (via code)

Jika ingin menambahkan data secara manual atau reset data:

```dart
import 'package:perpusku/utils/seed_data.dart';

// Tambah semua data
await SeedData.seedAll();

// Atau tambah per kategori
await SeedData.seedBooks();
await SeedData.seedMembers();
await SeedData.seedTransactions();

// Hapus semua data
await SeedData.clearAll();
```
